# Remakist-Sans
Branding font for ZE's Remakes YouTube channel.

### WARNING:
This font does not have an official name yet and is subject to change in the final release of the font.

### About
Remakist Sans is the newest ZE's Remakes brand typeface, designed to be playful and fun, while keeping a uniform and consistent feel.

The font comes in a few variants, many of them which were planned for release.
- Remakist Sans Display (sans-serif font with uniform strokes);
- Remakist Sans Text (based on my handwriting).

### Software used
- Inkscape (for drawing glyphs);
- FontForge (for combining the glyphs and generating them into a functional font).
